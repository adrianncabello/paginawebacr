Cpu-Z / Rufus

Cpu-Z

1Primer paso

Instalar el setup de la pagina oficial de cpu-z.

2Segundo paso

Despues de ejecutarlo se instalaran un software minimo para detectar el hardware y ya estaría listo cpu-z para usarse, este programa se utiliza para saber el hardware que tiene el equipo, con todos los detalles.

Rufus

1Primer paso

Descargar la ultima version del ejecutable de rufus.

2Segundo paso

Al ejecutarlo ya podria usarse rufus para bootear un pendrive con un sistema operativo.
